## 加评论
在每条博客下面都可加评论，如需加图则按入评论框，其左下角会出一小方图，按它就可上传图像。目前只要留名即可加评论，如先行[登录](https://disqus.com/profile/login/)则所有个人评论都可集中管理。

## 加新帖
本站用 github 和 jekyll 制作，加贴者需[登录](https://github.com/login)，  
 1. 登录后按网页最下端中间“加新帖”链
 2. 填入帖名
 3. 嘮叨几句
 4. 按“commit changes"。
